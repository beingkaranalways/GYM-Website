## ✨ Features

- ⚡ **Animated Loading Screen** with percentage counter  
- 🎯 **Sticky Navbar** with active section highlight on scroll  
- 🌑 **Dark / Light Mode** toggle  
- 📊 **Scroll Progress Bar** at the top  
- 🖊️ **Typing Effect** in hero section  
- 🔢 **Animated Counters** (Members, Trainers, Champions)  
- 🎨 **Subtle Particle Background** — lightweight canvas animation  
- 📸 **Gallery with Lightbox** — click to expand, keyboard navigation  
- 💬 **Auto-play Testimonials Slider** with touch swipe support  
- 📬 **Working Contact Form** via [Formspree](https://formspree.io) — real emails!  
- 🛡️ **Honeypot spam protection** on the form  
- 📱 **Fully Responsive** — mobile, tablet, desktop  
- 🖼️ **Subtle Image Hover Animations** — zoom, light shine, top glow  
- 🔝 **Back to Top** button  
- 🪄 **AOS scroll animations** on all sections  
- 🔍 **SEO optimized** — meta tags, Open Graph, semantic HTML

## 📬 Contact Form Setup

The form uses **Formspree** to deliver messages to your email inbox.

1. Go to [formspree.io](https://formspree.io) and create a free account
2. Create a new form and copy your endpoint URL
3. Replace the URL in `index.html` and `script.js`:
   ```html
   action="https://formspree.io/f/YOUR_ID"
   ```
4. Done! Real emails will now arrive in your inbox ✅
